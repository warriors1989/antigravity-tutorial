# Antigrivity 极简教程

Google Antigrivity 提供了多种产品形态，覆盖不同的开发工作流。本项目围绕以下工具整理极简教程：

- **Antigrivity 2.0** — 桌面端指挥中心，支持项目管理、多工作区/工作树、并行子智能体编排。
- **Antigrivity CLI** — 轻量级终端界面，核心智能体能力直达命令行，适合快速交互与 SSH 会话。
- **Antigrivity SDK** — Python 编程框架，提供声明式安全策略、生命周期钩子和编程式子智能体生成。
- **Antigrivity IDE** — 全功能 AI 驱动开发环境，深度集成编码智能体、MCP、Skills 等能力。
